# 🧠 DeepSeek 记忆助手

> 一个浏览器脚本，让 DeepSeek 记住你是谁、你在做什么、你的常用指令。
> 
> **适合**：英语老师、内容创作者、任何需要反复告诉 AI "我是谁" 的人。

---

## ✨ 功能一览

| 功能 | 说明 |
|------|------|
| 👤 **用户记忆** | 告诉 DeepSeek 你的身份、风格、偏好，一次填写，永久复用 |
| 📁 **项目管理** | 创建不同项目（如"KET讲义""暑假班"），每个项目独立记忆 |
| 📍 **当前进度** | 记录正在做的事，一键标记完成 |
| 📋 **已完成** | 自动归档已完成的任务，形成工作日志 |
| 📝 **项目核心记忆** | 存储项目规范、要求、素材，随时调用 |
| 🎯 **快捷模板** | 一键复制常用指令（如"讲义制作""自查清单"），不用反复打字 |
| 📋 **复制全部** | 一键合并所有记忆，粘贴到 DeepSeek 对话框 |

## 🚀 快速安装

### 第一步：安装暴力猴（Violentmonkey）

暴力猴是一个浏览器扩展，用来运行用户脚本。

- **Chrome**：从 [Chrome 网上应用店](https://chrome.google.com/webstore) 搜索 "Violentmonkey" 安装
- **Edge**：从 [Edge 外接程序](https://microsoftedge.microsoft.com/addons) 搜索 "Violentmonkey" 安装
- **Firefox**：从 [Firefox 附加组件](https://addons.mozilla.org) 搜索 "Violentmonkey" 安装

装好后，浏览器右上角会出现 🐵 图标。

### 第二步：安装记忆助手

点击下面的链接，暴力猴会自动弹出安装窗口：

👉 **[点击安装记忆助手](https://github.com/Jessie42140/deepseekmemory/raw/main/ds_memory.user.js)**

点击「安装」，刷新 DeepSeek 页面，右下角会出现 🧠 橙色小圆球。

## 📖 使用指南

### 1. 填写用户记忆

点击 🧠 打开面板，在「👤 用户记忆」框里告诉 DeepSeek 你是谁：

```
我是 Jessie，13 年广州初中英语老师
教全年龄段，擅长 KET/PET 备考
不喜欢蓝色，偏好墨绿/深灰/暖棕色系
```

点击「保存」。

### 2. 创建项目

在「📁 项目」下拉菜单选择 **+ 新建项目**，输入项目名称，如：
- `KET Part 1 讲义`
- `暑假班备课`
- `公众号文章`

### 3. 记录进度和记忆

- **📍 当前进度**：填写正在做的事，如 `⏳ 写 KET Test 3 解析`
- **📝 项目核心记忆**：填写项目规范，如排版要求、字数限制、风格指南
- 点击 **💾 保存项目进度**

### 4. 使用模板

在「🎯 快捷模板」区域点击 **+ 添加**，创建常用指令：

**示例模板**：
- 名称：`📝 讲义制作`
- 内容：`请按以下规范制作讲义：1. 四部分结构 2. 5道GC题 3. 核心词汇表`

以后点击模板按钮，内容自动复制到剪贴板，粘贴到 DeepSeek 即可。

### 5. 一键复制全部记忆

点击 **📋 复制全部**，所有记忆（用户记忆 + 项目进度 + 核心记忆）合并复制，粘贴到 DeepSeek 对话框，AI 立刻知道上下文。

## 🛠️ 高级功能

### 标记任务完成

当当前进度里的任务做完后，点击 **📋 标记完成**，系统会自动：
1. 把当前任务移到「已完成」列表
2. 当前进度重置为 `⏳ 新任务（请编辑）`

### 多项目管理

随时切换不同项目，每个项目有独立的进度、已完成列表、核心记忆。适合同时推进多个备课/写作任务。

### 数据安全

所有数据只保存在你自己的浏览器本地存储中，**不上传任何服务器**。换电脑或清除浏览器数据会丢失，建议定期备份。

## ❓ 常见问题

**Q：安装后没有出现 🧠 图标？**
> 刷新 DeepSeek 页面，或检查暴力猴是否已启用该脚本。

**Q：数据存在哪里？会泄露吗？**
> 数据只存在浏览器本地，不上传服务器。但换电脑会丢失。

**Q：能手机用吗？**
> 目前只支持电脑浏览器。手机可用 Kiwi Browser 安装暴力猴，但体验有限。

**Q：怎么卸载？**
> 点击 🐵 暴力猴图标 → 找到 "DeepSeek 记忆助手" → 删除。

**Q：更新脚本后数据会丢吗？**
> 不会。数据独立存储，更新脚本不影响。
